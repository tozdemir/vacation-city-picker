Column format rules

The backend prompt enforces coherent values:

  • Yes/No columns use exactly Yes, No, Nearby, or Seasonal where appropriate. No TRUE/FALSE.
  • Scores are numbers from 0 to 10.
  • Cost fields use short text like 180-320 EUR pp.
  • Lists use semicolons, e.g. Aquarium; Science Museum; Beach.
  • Weather uses compact text, e.g. 20-24C, mild autumn.
