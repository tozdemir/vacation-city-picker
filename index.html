Google Sheets structure

Use one row per city. Required minimum columns are city and preferably country. Everything else is optional and becomes a comparison factor automatically.

Recommended columns:

city, country, overall, weather, beach, flight, flight_cost, hotel_estimate, child_activities, sightseeing, day_trips, food, walkability, rain_risk, pros, cons, ideal, notes

Publishing steps

  1. Google Sheets → File → Share → Publish to web.
  2. Select the sheet tab and choose Comma-separated values (.csv).
  3. Copy the published CSV URL.
  4. Paste it into this page and click Save.
  5. Upload this index.html to GitHub Pages / Netlify / Vercel.

Making it more sophisticated later

  • Add numeric columns like flight_score, hotel_score, child_score, weather_score.
  • Add a weighted scoring model with sliders.
  • Add a second sheet for itinerary ideas per city.
  • Add live flight-price fields manually or via an external API/backend.
Tip: Keep the Google Sheet public/published only if the contents are safe to share. Published CSV links are effectively public.
